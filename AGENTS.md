# AGENTS.md

## 目的
このドキュメントは、このリポジトリで作業するすべての AI エージェント（汎用エージェントを含む）に向けた共通の作業方針とルールを定義します。

## 基本方針
- **会話言語**: 日本語を使用してください。
- **コメント言語**: コード内のコメントは日本語で記述してください。
- **エラーメッセージ**: 英語で記述してください。
- **コミット規約**: [Conventional Commits](https://www.conventionalcommits.org/en/v1.0.0/) に従ってください。description は日本語で記述します。
- **フォーマット**: 日本語と英数字の間には半角スペースを入れてください。

## 判断記録のルール
作業中に重要な判断を行った場合は、以下の内容を明確に記録または説明してください：
1. 判断の内容
2. 検討した代替案
3. その案を採用した理由
4. 前提条件や不確実性

## 開発手順（概要）
1. **プロジェクト理解**: `README.md` やソースコードを読み、目的と構造を理解します。
2. **依存関係**: `pip install -U -r requirements.txt` でインストールします。
3. **実装**: 既存のコードスタイル (`flake8` 準拠) に合わせて変更を行います。
4. **検証**: `flake8` による Lint チェックと、可能な範囲での動作確認を行います。

## セキュリティ / 機密情報
- **認証情報**: `DISCORD_TOKEN` などの認証情報は絶対にコミットしないでください。
- **ログ**: ログファイルに認証情報や個人情報を出力しないように注意してください。

## リポジトリ固有
- このツールは Systemd の状態を監視するため、Linux 環境での動作が前提となっています。
- `notified_ids.json` や `logs/` ディレクトリは実行時に生成・使用されます。これらは `.gitignore` に追加されており、コミットされません。
